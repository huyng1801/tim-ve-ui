# Easy Room Rental Demo UI

Demo Flutter app thuê phòng với dữ liệu giả, không cần backend.

## Chạy ứng dụng

```bash
cd d:\Outsourcing\Node.js\tim-ve-ui
flutter pub get
flutter run
```

Chạy trên web:

```bash
flutter run -d chrome
```

## Tài khoản demo

Thông tin đăng nhập đã được điền sẵn trên màn hình Login:

## Tính năng demo

- Splash → Login/Sign Up (pre-filled)
- Trang chủ: danh sách phòng, filter, banner ưu đãi
- Tìm kiếm & lọc theo thành phố, loại phòng, giá
- Chi tiết phòng: gallery, tiện ích, đánh giá
- Đặt phòng demo với chọn ngày
- Yêu thích (toggle local)
- Thông báo, Chat, Hồ sơ
- Đăng xuất về màn Login
